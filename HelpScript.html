WXScript

什么是WXScript?

WXScript是集成在WinXetor工具里面的一种脚本语言

主要用于方便脚本调用WinXetor的API与丰富功能模块

如果你想单独编写一个脚本可以参考此处

下载独立版本

指令格式

_off.echo.console // 指定文件运行参数


_api.cout // 调用API

Hello World // 指定内容

_return.backmain // 退出调用的API


_wxscript.close // 退出脚本

注意 // 只是用于解释指令，如果你想在脚本中使用注释请使用 _skip

基本指令操作

基于 Version 1.02 / 版本代号 : 2 编写


_off.echo.console // 如果你不想看到调试信息可以脚本头处添加此指令

_set.min.version // 规定执行脚本要求的WXScript核心最低版本

_use.admin.run // 如果想让此脚本以管理员权限运行请添加此指令

_wxscript.close // 结束返回程序指令

_skip // 跳过执行下一段指令用于添加注释


用户交互

_wxscript.pause // 暂停脚本执行

_wxscript.clear // 清理控制台

_print.TXT // 输出TXT文件

注意：返回时需要使用 _return.backmain 指令，否则视为继续执行
